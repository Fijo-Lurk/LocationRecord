## installation

- Clone the repo

  ```sh
  $ git clone https://github.com/Fijo-Lurk/LocationRecord.git
  ```

- Install and run frontend

  ```sh
  $ cd frontend
  $ npm install
  $ cd ../
  $ npm run start:frontend
  ```

- Install and run server
  ```sh
  $ cd backend
  $ npm install
  $ cd ../
  $ npm run start:server
  ```
